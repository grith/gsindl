3. certficate – Handle certificates and requests¶

3.1. generate_request()¶

gsindl.certificate.generate_request(dn=None, request=None, path=None, keySize=2048, key=None, extensions=None)¶

This funciton is for certificate request generation.

Parameters:
  • request – either a PEM str a DER str
  • path – the path to the certificate request file
  • key – if sepecified this will be wrapped in a Key
  • keySize – The size of the key to be generated (default 2048)
  • dn – the DN string of M2Crypto X509_Name
  • extensions – a list of dict objects containing extensions
