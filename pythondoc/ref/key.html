5. key – Handle key pairs¶

5.1. generate_key()¶

gsindl.key.generate_key(key=None, keySize=2048, callback=<function no_passphrase_callback at 0x388ab90>)¶

This is a wrapper class for handling key pair generation.

Parameters:
  • key – the str or file path to the key
  • keySize – The size of the key to be generated (default 2048)
  • callback – a function that is called when outputting the key, it’s used to encrypt the key before writing it.
